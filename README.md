# Mivon HTML Editor
A powerful WordPress plugin that turns any folder ending with `-templates` into a dynamic, shortcode-driven HTML theme system with a live editor, asset auto-mapping, and a theme manager with toggle switches.

Version: **2.1.1**  
Author: **Stefan Gold**
